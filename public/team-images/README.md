# Team Logos/Images

This folder stores team logos and images uploaded through the admin panel.

## Supported Formats
- PNG
- JPG/JPEG
- GIF
- WEBP

## Size Limit
- Maximum file size: 5MB

## Usage
Images are automatically uploaded when adding or editing teams in the admin panel. They are displayed:
1. In the team management cards (50px circular logos)
2. In the auction view team list (40px circular logos)
3. In the bid panel as clickable team buttons (45px circular logos)

## Naming Convention
Team images are saved with the `team_` prefix followed by the sanitized team name:
- Example: `team_mumbai_indians.jpg`
- Example: `team_chennai_super_kings.png`
