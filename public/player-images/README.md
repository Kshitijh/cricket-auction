# Player Profile Images

This folder stores player profile pictures uploaded through the admin panel.

## Supported Formats
- PNG
- JPG/JPEG
- GIF
- WEBP

## Size Limit
- Maximum file size: 5MB

## Usage
Images are automatically uploaded when adding or editing players in the admin panel. They are displayed:
1. In the player table (40x40px thumbnail)
2. In the player card during bidding (60x60px circular frame)
