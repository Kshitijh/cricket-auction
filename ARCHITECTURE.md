## Key Design Patterns Used

1. **Provider Pattern** - AuctionContext wraps entire app
2. **Container/Presentational** - Smart vs. Dumb components
3. **Composition** - Building complex UIs from simple components
4. **Custom Hooks** - useAuction() for state access
5. **Reducer Pattern** - Centralized state updates
6. **Controlled Components** - Forms with React state
7. **Conditional Rendering** - Dynamic UI based on state
8. **Event Delegation** - Efficient event handling
